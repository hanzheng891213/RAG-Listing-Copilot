# 商品详情页规则 (Product Detail Page Rules)

## 概述

Amazon 对商品详情页有严格规范，涵盖标题、图片、五点描述、A+页面和变体写法。违反详情页规则可能被判定为"详情页滥用"(Detail Page Abuse)，导致 Listing 被移除甚至账户停用。

---

## 商品标题要求 (Title Requirements)

| 规则 | 说明 |
|------|------|
| **最大长度** | 200 个字符（含空格），Amazon 建议控制在 80 字符以内 |
| **推荐格式** | 品牌 + 商品名称 + 核心特性 + 尺寸/数量 |
| **大小写** | 使用正确的大小写，禁止全大写（ALL CAPS）|
| **数字格式** | 使用阿拉伯数字 "2" 而非 "two" |
| **禁止内容** | HTML 标签、特殊符号（! * $ ?）、促销短语如"免费送货"、"100%品质保证" |
| **禁止信息** | 卖家名称、URL、营销信息 |
| **纯文本** | 不能包含任何 HTML 格式 |

### 标题示例
- ✅ 正确：`XYZ Brand Wireless Bluetooth Headphones - Noise Cancelling, 40H Battery Life, Black`
- ❌ 错误：`***BEST*** XYZ Headphones!!! Free Shipping!!! BEST DEAL ON AMAZON!!!`

---

## 主图要求 (Main Image Requirements)

| 规则 | 说明 |
|------|------|
| **背景** | 纯白色（RGB 255, 255, 255） |
| **分辨率** | 至少 1000 x 1000 像素（推荐最长边 1600+ 像素以启用缩放功能）|
| **产品占比** | 产品必须占画面至少 85% |
| **展示内容** | 仅展示待售商品，不包含非附赠的配件或道具 |
| **禁止内容** | 无文字叠加、无水印、无边框、无 Logo、无 URL |
| **禁止风格** | 无彩色背景、无图形插图、无动画图片 |
| **禁止元素** | 无模特/人体模型（生活方式图例外）、无线条图 |
| **格式** | JPEG（推荐）、TIFF (.tif) 或 PNG (.png) |
| **每图展示** | 每张图只展示一个产品（不要在一张图中展示多个颜色/视角）|

### 常见图片违规
- 非纯白背景：产品放在彩色或灰色背景上
- 产品占比不足：产品太小，留白过多
- 图片上有促销文字："50% OFF"、"Best Seller" 等
- 包含不随产品出售的配件
- 主图使用生活方式图（非白底场景图）

## 五点描述要求 (Bullet Points Requirements)

| 规则 | 说明 |
|------|------|
| **数量** | 3-5 条（部分类别要求恰好 5 条）|
| **每条长度** | 最多 500 个字符 |
| **内容导向** | 侧重于产品优势 (Benefits) 而不仅是特性 (Features) |
| **格式** | 每条以大写字母开头 |
| **禁止内容** | 无促销语言、无定价信息、无库存信息、无配送信息 |
| **禁止元素** | 无 HTML、无引号、无客户评价、无评论征集 |

### 最佳实践
- 每条解决客户的一个痛点
- 突出产品解决什么问题
- 包含使用场景
- 用具体数据和比较（如 "比标准版轻 30%"）
- 将最重要的信息放在第一条

---

## 产品描述要求 (Product Description Requirements)

| 规则 | 说明 |
|------|------|
| **最大长度** | 2000 字符（部分类别有所不同）|
| **允许的 HTML** | 仅允许 `<br>` 换行标签 |
| **禁止内容** | 无促销文案、无定价、无配送信息 |
| **禁止元素** | 无 JavaScript、无其他脚本、无外部链接 |
| **内容要求** | 全面的产品信息，帮助客户做出购买决定 |

### A+ 内容 (Enhanced Brand Content / A+ Content)
- 仅限品牌注册卖家使用
- 包含增强图片、对比表、品牌故事模块
- 必须遵守 A+ 内容规范（无促销声明、无保证/保修声明、无链接）

---

## 变体规则 (Variation Rules)

| 规则 | 说明 |
|------|------|
| **有效变体主题** | 颜色、尺寸、样式、口味等（需在 Amazon 批准的变体主题内）|
| **禁止行为** | 将不相关商品绑为变体、将不同款式产品绑在一起 |
| **违规后果** | 被判定为 "变体滥用"，Listing 可能被拆分或移除 |
| **要求** | 父商品下所有子变体必须本质上是同一产品 |

### 常见变体违规
- 将不同款式/功能的产品绑在一起
- 使用不正确的变体主题（如将颜色差异设为尺寸变体）
- 在变体标题中添加不一致的信息

---

## 搜索词要求 (Search Terms / Backend Keywords)

| 规则 | 说明 |
|------|------|
| **最大长度** | 250 字节（不是字符）|
| **重复规则** | 不要重复标题中已有的词 |
| **分隔符** | 用空格分隔，不需要逗号 |
| **禁止内容** | 无品牌名（除非是你的品牌）、无 ASIN、无临时性声明 |
| **不区分大小写** | 系统自动处理 |

---

## 通用详情页规则

| 规则 | 说明 |
|------|------|
| **禁止代码** | 详情页不得包含 HTML、JavaScript 或其他代码（除 `<br>` 外）|
| **禁止内容** | 无淫秽/冒犯性材料、无电话号码、无邮件地址、无 URL |
| **禁止信息** | 无时效性信息（巡展日期、研讨会等）|
| **禁止促销** | 无交叉商品推销或交叉推广 |
| **类目准确性** | 产品必须准确归入正确的浏览树类目 |
| **杜绝重复** | 不得为 Amazon 目录中已有产品创建重复页面 |
| **国际产品** | 国际产品必须在正确的国家站点详情页上架 |

---

## 违规后果

| 违规类型 | 处罚 |
|---------|------|
| 轻度违规（标题/图片不规范）| Listing 被抑制或移除 |
| 中度违规（详情页滥用）| Listing 移除 + 账户警告 |
| 严重违规（变体滥用、重复创建）| 账户权限受限或停用 |
| 多次违规 | 销售权限被永久移除 |

- [ ] 无文字叠加

- [ ] 每条 ≤ 500 字符

- [ ] 无促销语言

- [ ] 无 HTML（除 `<br>` 外）

- [ ] 无促销信息
