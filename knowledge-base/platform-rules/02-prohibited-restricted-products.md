# 禁售与受限商品政策 (Prohibited & Restricted Products)

## 概述

Amazon 将商品分为三个层级：完全禁售（无批准途径）、受限/需批准（Gated Categories）、自由销售。各站点的清单略有差异，跨境销售时必须逐一核实目标站点的具体政策。

---

## 完全禁售商品（无论如何都不能销售）

以下商品类别在 Amazon 上**没有批准途径**，一经发现将导致立即账户停用：

### 违禁品与不安全物品
- 处方药（持牌药房除外）
- 管制物质和毒品
- **假冒伪劣商品**（零容忍）
- 赃物和非法商品
- 被召回的产品（CPSC 或其他监管机构的强制召回）

### 武器与爆炸物
- 枪支及配件
- 弹药和爆炸物
- 自动刀具、弹簧刀
- 电击枪/泰瑟枪（因站点而异）

### 成人内容
- 色情材料
- 成人视频和杂志
- 性玩具和成人用品

### 烟草与酒精
- 香烟、雪茄、烟斗烟草
- 酒精饮料（仅持牌分销商可销售，大多数卖家无法通过审核）

### 危险品
- 易燃液体
- 压缩气体
- 放射性材料
- 腐蚀性物质

---

## 受限商品类别（Gated Categories — 需批准）

以下类别可销售，但必须先通过 Amazon 审核批准：

| 类别 | 关键要求 |
|------|---------|
| **收藏品与奢侈品** | 硬币、艺术品、珠宝、手表、体育纪念品、邮票 |
| **媒体与电子产品** | 音乐/DVD、流媒体播放器、蓝光 |
| **婴儿产品** | 喂养用品、活动装备、婴儿车/背带 — 需 CPC 安全认证 |
| **健康与个人护理** | 膳食补充剂、医疗用品、CPAP 设备 |
| **汽车与动力运动** | 合规图片、翻新件需提供保修 |
| **美容外用产品** | 高仿冒风险类别 — 需品牌授权 |
| **珠宝** | 高级珠宝需 1 年以上销售经验、$50K 年收入、实体店 |
| **食品与饮料** | 经批准的供应商发票、FDA 合规 |
| **非处方药** | FDA 注册、GMP 合规、COA |
| **宠物食品与护理** | 可能需要批准 |

## 申请解除限制（Ungating）流程

### 基本卖家要求
- **专业卖家账户**（$39.99/月）— 个人卖家账户无法销售受限商品
- 良好的绩效指标：
  - 订单缺陷率 (ODR) < 1%
  - 发货前取消率 < 2.5%
  - 延迟发货率 < 4%
- 公司网站（展示合法性）

### 通常需要的文件
- **发票**：开具日期在 90-180 天内，打印版（非手写），信息与 Seller Central 匹配
- **供应商资质**：供应商必须是制造商或授权分销商
- **品牌授权书 (LOA)**：适用于品牌商品
- **安全认证**：CE（欧盟）、FCC（美国）、CSA（加拿大）
- **第三方实验室检测报告**：适用于受监管商品类别

### 申请步骤
1. Seller Central → 库存 → 添加商品
2. 搜索想要销售的商品
3. 查看状态指示：
   - **"销售此商品"** — 无需批准
   - **"申请销售"** 或 **"Listing 限制适用"** — 需要批准
   - **"不可用"** — 无法上架
4. 点击"请求批准"
5. 提交所需文件
6. 等待 1-7 个工作日

## 禁止的声明与关键词 (Prohibited Claims & Keywords)

在 Listing 中严格禁止以下声明和关键词：

### 健康/医疗声明（执行最严格）
- 疾病名称：癌症、COVID-19、糖尿病、HIV 等
- 治疗语言："治愈"、"治疗"、"愈合"、"药物"
- 杀虫/抗菌："抗菌"、"抗微生物"、"病毒"、"消毒"
- 减肥/健康："排毒"、"生酮"、"减肥"（未经 FDA 批准）

### Listing 文案中禁止
- **最高级词汇**："最好的"、"第一"、"最高评价"、"获奖"
- **促销/定价用语**："免费送货"、"限时优惠"、"清仓"
- **外部引用**：URL、电话号码、邮箱、二维码
- **知识产权违规**：未经授权的品牌名称、竞争对手引用
- **R/™ 符号**：在面向客户的内容中使用会导致 ASIN 被抑制

## 各站点差异

| 商品 | 美国 | 英国 | 加拿大 |
|------|------|------|--------|
| 婴儿学步车 | ✅ 允许 | ✅ 允许 | ❌ 2021年起禁止 |
| 婴儿睡眠定位器 | ❌ CPSC 禁用（SIDS 风险） | ✅ 允许 | ❌ 禁用 |
| 抗菌产品 | ❌ 可能被归类为杀虫剂 | ✅ 允许 | ⚠️ 需检查 |
| 电子产品 | FCC 认证 | CE 认证 | CSA 认证 |

---

## 违规后果

| 违规次数 | 处罚 |
|---------|------|
| 首次违规 | Listing 移除 + 警告 |
| 二次违规（30-60天内） | 账户停用 |
| 严重违规（假冒、武器等） | 永久关闭账户 + 资金冻结 |
| 账户停用申诉 | 7-10 天内提交行动计划 (POA) |

---

## Listing 前检查清单

1. ✅ 检查目标站点的类别是否受限
2. ✅ 验证品牌级限制（Nike、LEGO、Disney、Samsung 等大品牌常有额外门槛）
3. ✅ 确认各国的具体限制（跨境销售时尤其重要）
4. ✅ 准备合规文件：发票、认证、LOA
5. ✅ 审查禁止声明 — 无疾病名称、无最高级、无未经证实的"天然"声明
6. ✅ 检查产品召回状态（美国 CPSC、英国 OPSS）
7. ✅ 确保 Listing 图片合规 — 无徽章、无二维码、无水印
